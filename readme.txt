- Anh em nào phụ trách web thì nhớ thêm ảnh vào folder "img" kia (hướng dẫn đã có ở trong folder), xong thì mở "index" để kiểm tra ảnh của anh em đã lên chưa.
- Để thay đổi title, ae mở "index" trong Visual Studio Code (hoặc Notepad, blah blah), sau đó tìm dòng "content__title no-select" để sửa tiêu đề.
- Để căn chỉnh ảnh, anh em mở "style" trong Visual Studio Code (hoặc Notepad, blah blah), sau đó tìm "/* Shorthand grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end */" để căn chỉnh ảnh.
